# Tic Tac Toe - AI with Minimax Algorithm

This project implements an unbeatable Tic Tac Toe AI using the minimax algorithm.

## Files
- `runner.py` - Game runner and UI
- `tictactoe.py` - AI logic and game mechanics

## How to Run
```bash
python runner.py
```

## Minimax Algorithm
The AI uses the minimax algorithm to evaluate all possible game states and choose the optimal move.
