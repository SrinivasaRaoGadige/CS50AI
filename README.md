# CS50 AI Projects

A collection of AI and computer science projects from CS50's Introduction to Artificial Intelligence with Python.

## Repository Structure

```
CS50AI/
├── crossword/           # Crossword puzzle solver
├── degrees/             # Social network analysis using BFS
├── heredity/            # Heredity prediction using Bayesian networks
├── knights/             # Knights puzzle solver using backtracking
├── minesweeper/         # Minesweeper game with AI solver
├── nim/                 # Nim game with minimax AI
├── pagerank/            # PageRank algorithm implementation
├── parser/              # Natural language parser
├── questions/           # Question answering system using TFIDF
├── shopping/            # Product recommendation system
├── tictactoe/           # Tic Tac Toe with minimax algorithm
└── traffic/             # Traffic sign recognition with neural networks
```

## Projects

### 1. **Degrees** (`degrees/`)
Investigates the "degrees of separation" between two actors using a breadth-first search (BFS) algorithm to find the shortest path through a movie database.

### 2. **Tic Tac Toe** (`tictactoe/`)
An unbeatable Tic Tac Toe AI implementation using the minimax algorithm.
- **How to run:** `python runner.py`
- **Requirements:** See `requirements.txt`

### 3. **Minesweeper** (`minesweeper/`)
An AI-powered Minesweeper solver using constraint satisfaction and logical inference.

### 4. **Knights** (`knights/`)
Solves the Knights puzzle using backtracking and constraint satisfaction.

### 5. **Nim** (`nim/`)
Implements the Nim game with an AI opponent using the minimax algorithm.

### 6. **Pagerank** (`pagerank/`)
Implements the PageRank algorithm to rank web pages by importance.

### 7. **Heredity** (`heredity/`)
Uses Bayesian networks to predict hereditary traits.

### 8. **Parser** (`parser/`)
A natural language parser that analyzes sentence structure.

### 9. **Questions** (`questions/`)
A question answering system using TF-IDF for document retrieval.

### 10. **Shopping** (`shopping/`)
A product recommendation system using machine learning.

### 11. **Crossword** (`crossword/`)
A crossword puzzle solver using constraint satisfaction.

### 12. **Traffic** (`traffic/`)
Traffic sign recognition using convolutional neural networks.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/SrinivasaRaoGadige/CS50AI.git
   cd CS50AI
   ```

2. Navigate to a project:
   ```bash
   cd tictactoe
   ```

3. Install dependencies (if needed):
   ```bash
   pip install -r requirements.txt
   ```

4. Run the project according to its specific instructions.

## Technologies Used

- **Python 3.x**
- **Algorithms:** BFS, DFS, Minimax, Backtracking, PageRank, Bayesian Networks, TF-IDF
- **Data Structures:** Graphs, Trees, Sets, Queues
- **Machine Learning:** Neural Networks (for traffic sign recognition)
